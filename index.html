Reglas personalizadas

Edita cómo se clasifican los eventos, centros y ausencias. El formato es JSON editable.

{
  "centros": {
    "gines": ["(g)"],
    "bormujos": ["(b)"],
    "privado": ["(p)"]
  },
  "pattern_sesion": "^\\*?\\s*(\\d{3,6})\\s+(.+?)\\s*\\(([BGP])\\)\\s*$",
  "otros_excluir_totalmente": [
    "^g\\s*:\\s*\\d+$",
    "^b\\s*:\\s*\\d+$",
    "^ent\\.\\s*[bg]\\s*:\\s*\\d+"
  ],
  "bloques_horario": [
    "^[bg]\\s*\\d{1,2}$"
  ],
  "otros_keywords": [
    "firma",
    "comida",
    "coordinación",
    "reunión",
    "formación",
    "llamada",
    "administrativo",
    "battelle"
  ]
}

Usa * al inicio del título para marcar una ausencia.

La aplicación únicamente solicita acceso de lectura a tu Google Calendar. Conecta tu Client ID para empezar.
